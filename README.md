# Excel-project
Vrinda Store Sales and Performance Analysis Using ExcelIt focused on identifying sales trends, customer behavior, and performance gaps to support better business decisions and reduce manual reporting efforts.
Cleaned and prepared data by removing duplicates, handling missing values, and standardizing formats in Excel. Performed Exploratory Data Analysis (EDA) using pivot tables, charts, and filters to analyze sales trends and customer behavior.
Cut report preparation time from hours to minutes using automated Excel dashboards.
Enabled faster decision-making by providing clear visual insights into sales and customer trends.
